# Changelog

## 0.1.0

- Initial local prototype release under the UI Trace name.
- Chrome extension for capturing rendered webpage tokens, assets, components, containers, interaction-state evidence, and optional screenshot references.
- Figma development plugin for importing normalized JSON into Designer Kit, Component Library Draft, Page Screenshot Reference, and Audit Draft outputs.
- Node tools for merge, normalize, audit, validation, and local release packaging.
- Documentation for installation, usage, limitations, and distribution.
